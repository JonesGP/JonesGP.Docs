MKDocs

Para installar o mkdocs

pip install mkdocs

poetry add mkdocs

Existem basicamente dois tipos de arquivos no mkdocs, o .md que vão sempre esta na pasta /docs e mkdocs.yml que é o arquivo principal da pagina

CLI

para criar um novo projeto

mkdocs new <nome_do_projeto>

para colocar um servidor no ar para vermos a documentação no browser

mkdocs serve

para fazer o deproy no github pages

mkdocs gh-deploy

no mkdocs.yml tem o site_name que é o nome do proprio site

site_name: <nome do seu site>
