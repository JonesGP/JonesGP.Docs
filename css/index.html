Básico do CSS

Margin Collapse formas de tirar junção de margin

  • float: left / right;
  • position: absolute;
  • display: inline-block;

Propriedade Display

Por padrão, a maioria das tags vai vir com display: block. - display: block;: Significa que cada elemento vai ter sua própria linha, cancelando as propriedades de cada tag. - display: inline;: Cada elemento vai ficar na mesma linha, ignorando configurações de width e height. - display: inline-block;: Cada elemento fica na mesma linha, porém aceitando configurações de width e height.

Atenção: Comportamento estranho com inline-block no HTML vão ser considerados espaços em branco entre os itens. Em casos de uso do inline-block, se não carregar a imagem, a caixa inteira vai descer porque o tipo de alinhamento vertical é baseline. Para ajustar isso, é só colocar vertical-align: top; e não baseline.

Box Sizing

  • box-sizing: border-box;: Faz com que a caixa do elemento seja a caixa inteira, incluindo padding e border. Ou seja, se quiser que a da borda pra dentro seja considerado como tamanho que você pediu em width ou height.
  • box-sizing: content-box;: Faz com que a caixa seja do tamanho do próprio conteúdo. No caso, todos já vão vir com content box.

Overflow

  • overflow: hidden;: Esconde todo o overflow.
  • overflow: scroll;: Vai fazer o scroll tanto no eixo x quanto no y (scroll barra de rolagem).
  • overflow-x: scroll;: Vai fazer o scroll no eixo x.
  • overflow-y: hidden;: Vai esconder o overflow do eixo y.

Float

Float é a propriedade de flutuar tanto para a direita quanto para a esquerda, no centro e por aí vai. No entanto, float não é bom de usar. É melhor usar o flex box. - float: right;: Vai alinhar na linha a partir do primeiro elemento. - clear: right;: Serve para limpar o float anterior.

Unidade

  • Absolutos: px, cm, mm
  • Viewport: vh, vw, vmin, vmax
  • Font: rem (em relação à fonte raiz), em (em relação à fonte pai)

Position

  • position: static;: Valor padrão, apenas o espaço do conteúdo.
  • position: relative;: Posição relativa em relação ao elemento anterior.
  • position: absolute;: Posição absoluta em relação à página.
  • position: fixed;: O elemento fica fixo na página, independente da barra de rolagem.
  • position: sticky;: Utilize a propriedade top para a barra de menu.

Texto

  • font-style: italic;
  • font-weight: 100;
  • text-align: center;
  • font-transform: uppercase;
  • line-height: 2.5em;: Altura da linha.
  • text-decoration: overline underline line-through;
  • word-spacing: 10px;: Separação entre as palavras.
  • letter-spacing: 20px;: Separação entre as letras.

Valores padrão: - Unidades absolutas: - font-size: 100px; (pixels) - font-size: 72pt; (pontos) - font-size: 1in; (polegadas) - Unidades relativas: - font-size: 2.2em; (em relação ao tamanho da fonte pai) - font-size: 2.2rem; (em relação ao tamanho da fonte raiz) - font-size: 50vh; (em relação à altura da viewport)

Font-family: - font-family: monospace; (todas as letras ocupam o mesmo espaço) - font-family: Georgia, serif; (fonte serifada) - font-family: Verdana, sans-serif; (fonte não-serifada) - font-family: 'Indie Flower', cursive; (fonte cursiva)

Media Query

Usada para fazer sites responsivos, adaptando-se a diferentes aparelhos e tamanhos de telas.

Exemplo de uso:

@media (max-width: 600px){
    /* Código CSS aqui */
    .muito-pequeno {display: block;}
}

@media (max-width: 1000px), (orientation: portrait){
    /* Código CSS aqui */
    .pequeno-retrato {display: block;}
}

@media (min-width: 500px) and (max-width: 1000px){
    /* Código CSS aqui */
}

Flex Box

Quando você marca display: flex;, você transforma a caixa em um flex container.

  • display: flex;: Define o elemento como um flex container.

  • flex-direction: column or row;: Define a direção dos flex items dentro do flex container. Pode ser definido como row para alinhar os itens horizontalmente ou column para alinhar os itens verticalmente.

  • flex-wrap: wrap;: Quebra a linha dos itens de dentro quando não cabem em uma única linha.

  • flex-flow: row wrap;: Mistura a direção e a quebra de linha em uma única declaração.

Propriedades para ajustar os itens dentro do flex container:

  • justify-content: flex-start, flex-end, center, space-around, space-between;: Define como os flex items são distribuídos ao longo do eixo principal do flex container.

  • align-items: stretch;: Valor padrão que estica os itens para preencher todo o espaço disponível no eixo transversal.

  • align-items: baseline;: Alinha os itens com base na linha do texto.

  • align-content: flex-start, flex-end, center, space-around, space-between;: Alinha as linhas de itens quando há várias linhas no flex container.

Propriedades para os itens individuais:

  • order: 1, -1, 10, -877887,;: Define a ordem dos itens.

  • align-self: center;: Alinha um item selecionado, sobrescrevendo a propriedade align-items.

  • flex-grow: 0, 2, 3...;: Define como o espaço disponível é distribuído entre os itens quando há espaço extra.

  • flex-shrink: 1, 2, 3...;: Define como os itens encolhem quando há falta de espaço.

Outras propriedades relacionadas:

  • flex-basis: 100px;: Define a largura inicial de um item em relação ao eixo.

  • flex: 3 6 100px;: Propriedade de atalho que combina flex-grow, flex-shrink e flex-basis em uma única declaração.

Lembrando que a largura do item também pode ser definida pelo width ou height, mas a propriedade flex-basis é mais importante no contexto do Flexbox.### Flexbox

CSS Grid

  • display: grid;: define o elemento como um grid container.
  • grid-template-columns: 50% 50%;: cria duas colunas.
  • grid-template-columns: repeat(4, 8.33% 8.33% 8.33%);: repete as colunas especificadas 4 vezes.
  • grid-template-columns: 200px 20% 1fr;: define três colunas.
  • grid-template-columns: 3fr 6fr 3fr;: define três colunas proporcionais.
  • grid-template-rows: 50% 25% 12.5% 12.5%;: define quatro linhas proporcionais.

Propriedades para os elementos individuais:

  • grid-column: 2 / 5;: define que o item ocupa as colunas 2 a 5.
  • grid-row: 2 / 5;: define que o item ocupa as linhas 2 a 5.
  • grid-column-start: 2;: define que o item começa na coluna 2.
  • grid-column-end: 4;: define que o item termina na coluna 4.
  • grid-column-end: span 2;: define que o item ocupa duas colunas a partir da coluna atual.
  • grid-row-start: 3;: define que o item começa na linha 3.
  • grid-row-end: 5;: define que o item termina na linha 5.
  • grid-row-end: span 2;: define que o item ocupa duas linhas a partir da linha atual.
  • grid-area: 1 / 1 / 4 / 4;: define que o item ocupa as linhas 1 a 4 e as colunas 1 a 4.

Outras propriedades relacionadas:

  • grid-template-columns: [inicio] 1fr [metade-1 meio-1] 1fr [metade-2 meio-2] 1fr [fim];: define nomes para as colunas do grid.
  • column-gap: 20px;: define o espaçamento entre as colunas.
  • row-gap: 20px;: define o espaçamento entre as linhas.
  • gap: 20px 10px;: define o espaçamento entre as linhas e colunas.
  • grid-template-areas: 'nome-do-selector nome-do-selector' 'nome-do-selector nome-do-selector' 'nome-do-selector nome-do-selector';: define o layout do grid usando nomes para os seletores.

Animações CSS

  • transform: skew(-20deg);: inclina o objeto.

  • transition-property: color;: escolhe qual propriedade vai passar pela transição, podendo trabalhar com múltiplas propriedades.

  • transition-duration: 1s;: define a duração da transição para as propriedades especificadas.
  • transition-delay: 1s;: define o tempo de espera antes de iniciar a transição.
  • transition-timing-function: ease-in/ease-out/ease-in-out/cubic-bezier(.15,.89,.85,.26);: define a forma em relação ao tempo que a transição ocorrerá. Também é possível usar cubic-bezier() para personalizar o tempo de acordo com as curvas de Bezier.

Para animações usando keyframes, prefira usar opacity e transform:

  • animation-name: nome-da-animação;: define o nome da animação.
  • animation-duration: 1s;: define a duração da animação.
  • animation-timing-function: ease-inoutinout/cubic-bezier()/steps(3, jump-end/jump-start)/linear;: define a forma em relação ao tempo que a animação ocorrerá. Também é possível usar steps() para criar animações com saltos ou linear para um movimento linear.
  • animation-delay: 1s;: define o tempo de espera antes de iniciar a animação.
  • animation-iteration-count: 2.5/infinite;: define a quantidade de vezes que a animação será executada, podendo usar valores reais ou infinite para repetir infinitamente.
  • animation-direction: reverse/alternate/alternate-reverse;: define a direção da animação, podendo ser revertida, alternada ou alternada com reversão.
  • animation-fill-mode: forwards/;: define como os valores devem ser aplicados antes ou depois de uma animação. forwards mantém o último estado da animação, backwards retorna ao valor inicial mesmo que não seja definido no keyframe e both combina as duas opções anteriores.
  • animation-play-state: running/paused;: define o estado da animação, se está em execução ou pausada.
  • animation: fly 1s linear infinite alternate;: propriedade de atalho que combina várias propriedades relacionadas à animação.

Mantendo proporção em imagens

  • object-fit: cover;: faz com que a imagem não mude de proporção.
  • aspect-ratio: 16 / 9;: define a proporção da imagem.
