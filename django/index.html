para criar um novo projeto django-admin startproject

agora para criar uma aplicação dentro do nosso projeto django-admin startapp

1ª passo para fazela funcionar antes tem que importar a sua viewer do seu app que vai esta em views.py from import views uma viewer é basicamente uma função que determina oque fazer quando chegar naquela pagina

primeiro tem que criar uma rota para seu 'app' vai na pasta do seuprojeto e adiciona la em urls.py em urlpatterns = [ """ # rota, view resposavel, nome de preferencia # facebook.com path(''), colocando apenas espaços vazios vai ser para a pagina inicial # facebook.com/devaprender path('devaprender/') colocando o depois do indice inicial, vai para no caso facebook.com/devaprender """

path('', views.home, name='home') definir o nome é importante porque se voce quiser colocar um link para voltar para
a pagina inicial voce vai chamar o link pelo nome que definir aqui

]

2ª passo tem que criar uma viewer na views.py tem que deixar importando o from django.shortcuts import render a função home definida aqui sera a chamada em urls.py quando entrar na aplicação def home(request): request é um parametro que ja vem imbutido no django que te permite acessar os dados que estão dentro daquela pagina return render(request, 'usuarios/home.html') o render cria renderiza um pagina, e essa pagina precisa dos dados por isso request no incio, ai passa ai uma pagina html para exibir esses dados

no seu aplicativo, para criar htmls voce precisa criar um pasta templates e precisa ser exatamente nesse nome porque esse vai ser o nome que o django vai esta buscando é comum criar subpastas para organizar mais seus projetos no caso o arquivo home.html foi criado exatamente na pasta dentro do app templates\usuarios\home.html

depois disso tudo tem que cadastra nosso app dentro do nosso projeto quando voce criar um aplicativo ele precisa ser registrado dentro do seu projeto para fazer isso voce precisar acessar o settings.py do projeto em INSTALLED_APPS = ['nome da pasta do app']

ai sim voce pode rodar o servidor python .\manage.py runserver
