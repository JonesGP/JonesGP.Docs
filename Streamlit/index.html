StreamLit

para importar import streamlit as st (importar assim não é obrigatório)

Exibição de dados

para trabalhar com dados com o streamlit a gente pode usar varias ferramentas, mas a mais comum para trabalhar com dados massivos do python que é oque pessoas de ciencias de dados usam é o pandas pandas é uma ferramenta sensacial ela tras uma performance muito boa

para installar o pandas é pip install pandas para criar um dataframe que é um conjunto de dados do pandas import pandas as pd df = pd.DataFrame( o DataFrame vai criar um conjunto de dados com linhas e colunas de 2 dimenções #para gerar os dados vamos ultilizar o numpy para gerar um conjunto de dados, para trabalhar com algebra linear e matematica, para criar dados e vetores e matrizes então é uma ferramenta essecial para quem quer trabalhar com analise e ciencias de dados, então o numpy vai gerar aleatoriamnete os dados para trabalhar com o numpy tem que installar ele pip install numpy e para importar import numpy as np np.random.randn(10, 3) x, y linhas e columas columns=['Preço', 'Taxa de ocupação', 'Taxa de inadimplência', 'Pessoas por casa'] )

st.table(df) assim criamos uma tabela com os dados do pandas gerados pelo numpy sendo mostrada com o streamlit st.dataframe(df) a difereça do table para o dataframe é que o dataframe o usuario tem mais mobilidade de modificar a forma de visualização da tabela

st.line_chart(df) para criar um grafico de linhas top

st.bar_chart(df) para criar um grafico de barras

radios

opcao = st.radio( #dessa forma cria varias opçoes de radio 'Selecione uma opção', ('Opção 1', 'Opção 2') ) com as opções criadas voce pode criar varias condições para cada opção if opcao == 'opção 1': lembrando que a opção 1 ja vem selecionada st.header('Selecionei a opção 1') elif opcao == 'opção 2': st.header('Selecionei a opção 2')

selectbox

funciona de forma parecida com o radio porem muda a forma que é mostrado na tela como uma lista de opção para selecionar opcao = st.sidebar.selectbox( 'Selecione uma opção', ('Opção 1', 'Opção 2') )

multiselect

funciona de forma parecida como selectbox porem muda que da para o usuario selecionar mais de uma opção opcao = st.sidebar.multiselect( 'Selecione uma opção', ('Opção 1', 'Opção 2', 'Opção 3', 'Preço') ('Opção 1', 'Opção 3') passando esse campo no multiselect voce pode ja definir quais campos ja vão selecionados ) if 'Preço' in opcao: # se preço tinver na lista de opção selecionadas ele faz isso print(opcao)

slider

é aquele elemento que nos podemos arrastar para aumentar ou diminuir um slider assim definindo um maximo e um minimo opcao = st.sidebar.slider( 'Preço máximo e mínimo', 0.0, 100.0, (25.0, 75.0) aqui voce decide o minimo, o maximo, de entre parentes pode definir qual o valor que o slider vai começar )
