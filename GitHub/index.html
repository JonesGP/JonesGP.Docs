O que é o Git?

Git é um sistema de versionamento de codigo ou arquivos. Usado principalmente para criar um historico de versão para seu projeto

git --version versao do git

clear limpa o console

git restore --staged vai restaura os arquivos que estao na fila de espera

git init inicializa um repositorio vazio

git add manda os arquivos para area de estade, no caso chama os arquivos e os deixa esperando pra ser empurrado git add <.> vai mandar todos os arquivos para area de espera

git status vai mostrar

git commit -m manda os arquivos e da um titulo

git branch -M para renomear o branch

remote é a conexao local com o repositorio local do git hub add pra adicionar é o apelido do projeto git remote add https://github.com/JonesGP/ProjetoGIT.git aqui adiciona os arquivos no repositorios

git push -u main fazendo o push para o repositorio litealmente empurrando os aquivos que foram dados o commit

depois de ter conectado não precisa mais dar um remote para conectar com o servidor git push nao precisa mais do -u

para criar uma branc nova primeiro antes de mandar os arquivos tem que cria-la git checkout -b "nome da nova branch" usando tambem o checkout ele sai da branch atual e entra na nova criada

git checkout muda de branch

git merge aqui junta a branch com a principal desde que esteja na principal

git push agora depois disso tudo que manda os aquivos pra la

git clone link.git cria um clone de algum repositorio

git pull serve pra pegar alteraçoes do codigo

or push an existing repository from de the command line git remote add origin https://github.com/JonesGP/Platform-Prototipo_Godot.git git branch -M main git push -u origin main
